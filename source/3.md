Write 4 functions to perform addition, subtraction, multiplication, and division.
Each function, respectively *add*, *sub*, *mul*, and *div*, must accept 2 parameters (integers) *a* and *b*, and return the valid calculation.

![](assets/calc.png)

## Example
```python
>>> add(1, 2)
3
>>> sub(46, 4)
42
>>> mul(210, 2)
420
>>> div(8800, 2)
4400
```
